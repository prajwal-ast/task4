🛡️ Task 4 – Firewall Configuration using UFW
📌 Overview

Configured and tested a host-based firewall on Kali Linux using UFW (Uncomplicated Firewall). The objective was to understand firewall rule management by allowing and blocking network traffic through specific ports.

🛠️ Tools Used
Kali Linux 2025.3
UFW (Uncomplicated Firewall)
Linux Terminal

🎯 Tasks Performed
Installed and configured UFW.
Enabled the firewall.
Listed existing firewall rules.
Blocked Telnet (Port 23).
Allowed SSH (Port 22).
Verified firewall rules.
Removed the temporary test rule.

🔒 Outcome

Successfully configured and tested firewall rules using UFW. The exercise demonstrated how firewall policies filter network traffic, block insecure services such as Telnet, and allow secure remote access through SSH.
